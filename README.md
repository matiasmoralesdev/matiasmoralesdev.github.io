# Curriculum Interactivo - Matías Alberto Morales

Portfolio/curriculum interactivo construido con Vue 3, TypeScript y Vite, con modo oscuro y animaciones.

## Características

✨ **Interactividad**
- Modo oscuro / modo claro con persistencia
- Componentes modulares y reutilizables
- Modales animados para detalles de experiencia

🎨 **Diseño**
- Interfaz moderna y limpia
- Responsive en todos los dispositivos
- Animaciones suaves con Anime.js
- Variables CSS para temas

🔧 **Tecnología**
- Vue 3 con Composition API
- TypeScript para type safety
- Vite para build rápido
- Modular .vue components

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Preview de producción
npm run preview
```

## Estructura del Proyecto

```
src/
├── App.vue                 # Componente raíz
├── main.ts                 # Punto de entrada
├── style.css               # Estilos globales
├── components/
│   ├── Header.vue          # Cabecera con perfil
│   ├── Experience.vue      # Sección de experiencia
│   ├── ExperienceModal.vue # Modal de detalles
│   ├── Skills.vue          # Sección de habilidades
│   ├── Courses.vue         # Sección de cursos
│   ├── Education.vue       # Sección de educación
│   └── Footer.vue          # Pie de página
├── types/
│   └── index.ts            # Interfaces TypeScript
└── utils/
    └── theme.ts            # Utilidades de tema
```

## Scripts Disponibles

- `npm run dev` - Inicia servidor de desarrollo
- `npm run build` - Compila TypeScript y Vite
- `npm run preview` - Vista previa de producción
